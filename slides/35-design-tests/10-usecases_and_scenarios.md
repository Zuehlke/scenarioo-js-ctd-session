## Use Cases and Scenarios

Structure Scenarios by Use Case

* Use Case
  * A goal the user wants to achieve
  * Usually an action = verb and subject(s)
  * Not too fine granular! 
    * Boss Test
    * Coffee Break Test
    
* Scenario = Test Case = One flow through the use case
